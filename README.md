## Project Description
Krishi BD provides a user-friendly interface for managing agricultural activities.  
While the backend is partially implemented, all UI pages and interactions are complete and visually appealing.  
The project follows an MVC pattern with Controllers, Services, and Repositories.

---

## Features
- Home / Welcome Page  
- User Registration & Login  
- Dashboard after login  
- Plantation management (view/add plantations)  
- Product browsing  
- Shopping cart functionality  
- Trainee management  

---

## Technologies Used
- Java (JavaFX / Spring Boot or your framework)  
- Maven for build  
- FXML / HTML / CSS for UI  
- MySQL database (backend partially implemented)
